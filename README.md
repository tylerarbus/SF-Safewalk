# SF SafeWalk

SF Safewalk helps users plan a safe walking routes by mapping streets high in pedestrian relevant crimes.

## Usage

Run the following script from within the root directory:

```sh
npm run dev
```

## :package: Tech Stack

### Front-End

1. React
2. Google Maps Javascript API

### Back-End

1. Node
2. Express
3. MongoDB
4. SF Opendata API

### Development Tools

1. Webpack
2. Mocha
3. Enzyme
4. Supertest
